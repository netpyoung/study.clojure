# Boot

- 프로젝트 관리 프로그램이 있어야겠다. [boot](http://boot-clj.com/)를 사용하자.
- 새 프로젝트 생성은 [boot-new](https://github.com/boot-clj/boot-new)를 이용하자.

```bash
        boot -d boot/new new -t app -n sample
```
